# ZuluSCSI GitHub Organization page

ZuluSCSI is an advanced, high performance SCSI emulator, comprised of both firmware and hardware.

## Firmware

Visit https://github.com/ZuluSCSI/ZuluSCSI-firmware/releases/latest to access the latest and historical ZuluSCSI firmware releases.

## Development

ZuluSCSI firmware development primarily takes place at https://github.com/ZuluSCSI/ZuluSCSI-firmware, where you can also ask questions in the Discussion section, or report firmware issues in the issues section
